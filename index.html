How To Join 112c Network

Console

Method 1: Bedrock Together
Using Bedrock Together you can join the server on your preferred console.

Step 1:
Use a Android or iPhone or Chromebook (with Play Store)
Download Bedrock Together


Click Here For iPhone         Click Here For Android/Chromebook

Step 2: 
On Bedrock Together put the ip play.112c.co.uk and port 19132 then press the RUN button

Step 3:
Go to Multiplayer on Console then the server should show up!

PC

On PC you can get Java Edition for free if you already own Bedrock Edition.
 Simply download Minecraft Launcher from the Microsoft Store.

Java gives you access to Hyperworld's Custom Items and Custom Textures

If you still prefer to use Bedrock then go to the servers menu then press Add Server
IP: play.112c.co.uk PORT: 19132
